AR and Desktop app to visualise point in the spherical coordinate system

  • Created an application to visualize point in a spherical coordinate system using Unity.
  • This point is represented as position vector joining origin and the spherical coordinates and based on the parameters cone, sphere or point is generated. Unit vectors at that point is also shown on the app.
  • On specifying the velocity and acceleration of the point, the point is shown as a moving object along the 3D spherical coordinate system.
